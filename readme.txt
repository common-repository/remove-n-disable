A highly customizable plugin that will help to disable many links from dashboard.

== Description ==

This plugin has several options to disable/enable several links from wordpress dashboard, disable browser update warning, wordpress core update etc. This plugin's option will be found under settings named as R_n_D.<br />

You can modify below options by using this plugin <br />

= Features =

1)   Remove Wp Logo.
2)   Remove About Link.
3)   Remove Org Link.
4)   Remove Documentation Link.
5)   Remove support forum Link.
6)   Remove Feedback Link.
7)   Remove View Site Link.
8)   Remove Footer Version.
9)   Remove Footer Text.
10)   Disable Browser Upgrade Warning.
11)   Disable Wordpress Update.


== Installation ==

1. Upload `remove_n_disable` folder to the `/wp-content/plugins/` directory 
1. OR search for Top-authors in WP-admin > Plugins > Add New
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > R_n_D and set you own configuration.
4. Save settings.
